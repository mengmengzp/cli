#   fems-cli

#   功能

一键快速构建项目

# 安装依赖
 根目录执行：npm install
#   创建软链
   npm link

#   使用

打开你的cmd 输入fems或者fems -h 查看你可以使用的命令行

  Usage: fems "commands"


  Commands:
  
    create    通过命令直接创建vue项目  fems-create <项目名称>

  Options:

    -h, --help     output usage information
    -V, --version  output the version number
